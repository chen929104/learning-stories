# 选择排序

选择排序是一种排序算法，特别是就地比较排序。它具有 O（n2）时间复杂度，使其在大型列表上效率低下，并且通常比类似的插入排序更差。选择排序以其简单性着称，在某些情况下，特别是在辅助内存有限的情况下，它比更复杂的算法具有性能优势。
